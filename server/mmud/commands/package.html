Provides all the commands that are executable in the game. Commands can only be executed by characters.

The commands have an inheritance structure like this: 

      +----------+
      | Command  |  (Interface)
      +----------+
            |
    +---------------+ 
    | NormalCommand |
    +---------------+
            |
    +---------------+
    |  GuildCommand |
    +---------------+
            |
 +---------------------+
 |  GuildMasterCommand |
 +---------------------+
