Help on Boards

This functionality is used to display all available boards that are used for discussions in the game.

Right now it is not possible to make any changes. If the need is there, I will make it possible to add, delete or change boards.

It is possible to remove messages due to offensive content. This can be done by simply clicking the Remove link of the accompanying message. The message remains in the database, fully intact, but when reading a board, the message (and only the message) is replaced with the text [This message has been removed due to offensive content.].

Note: Bear in mind that the board called logonmessage is a special case used to display logonmessages to users when entering the game. This board should not be changed.
