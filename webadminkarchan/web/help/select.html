Help on Sql Select

The form can be used to enter either show or select sql statements that query the database. For example:

show tables

show fields from mm_usertable

select * from mm_usertable where name="Karn"

Note: Please use this responsibly. If you decide to make a query that shows all information of all users, you might end up with a really really large webpage. And you might drag the performance of the computer down along with it.

For more information on how the mmud database is set up, which tables contain which information in which fields, check here.

For more information on how to create sql statements, check the Documentation at the MySql Website.
