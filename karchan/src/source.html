Introduction

This page contains most of the introductory information necessary for you to get a similar Mud working like "Land of Karchan". I hope the information and files you need are present here, if not, or if you want to see some information added, gimme an email.

I give this source code along with all the other files freely available under the GNU Public Liscence (Also known as GPL, for more information do visit http://www.gnu.org.) Why do I give away the source code? Well, it's part of a new way of doing things. By keeping all source code private in the hands of companies you take away a piece of knowledge of which a lot of people can benefit. Try looking at it like instead of a company holding back the formula to the solution to a disease, having the company give out the formula for public use, thereby helping people all over the world. A lot of my source code is based upon other people's source code, who basically did the same thing as me, thereby creating a base on which I could build my mud.

The fact that I give away the source code does not mean that you can go ahead and make an exact duplicate of the Land of Karchan!!!

I'm giving you the source code of my mud in order for you to make something original with it. There is a clear distantiation between my mud (Maarten's MUD) and the Land of Karchan. While Maarten's Mud does fall under the GPL, Land of Karchan does no such thing and is copyrighted by me and others.

Have fun experimenting, and do send me an email if something is not clear.

REQUIREMENTS

There are a lot of requirements that need to be satisfied before the mud can be installed. If you are not really at home using UNIX like systems, and Linux, and a command line interface along with databases, perhaps you should reconsider downloading the source code to this mud.

Here's a list:

  • A computer (duh)
  • Linux (I personally use RedHat, but feel free to use any Distribution you want as I only provide the source code which can be recompiled at will) [www.linux.org]
  • gcc (Gnu C Compiler, usually already provided with Linux) [www.gnu.org]
  • Apache Web Daemon (cgi must be enabled as well as server-push) [www.apache.org]
  • cgic105 (a C library used to interpret World Wide Web Form Information) [www.boutell.com]
  • MySQL (the database, the client used for administration, and the development environment for compilation, so that's three packages:
    • MySQL-3.22.20a-1
    • MySQL-client-3.22.20a-1
    • MySQL-devel-3.22.20a-1
    [www.tcx.se]

For more information on installing the files, see the file INSTALL in the packages below.

  • mmud-1.4.1b.tar.gz, Maarten's Mud version 1.4.1beta
  • mmud-1.4.2a.tar.gz, Maarten's Mud version 1.4.2alpha
  • mmud-1.4.3.tar.gz, Maarten's Mud version 1.4.3
